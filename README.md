## 神秘鸭 golang 客户端
#### 一个面向高级玩家的神秘鸭客户端

**需要GUI，请走[GUI客户端](https://smya.cn/download "神秘鸭客户端")， 本程序不提供GUI！！**

**本程序只支持神秘鸭{执行任意命令}**

- 使用方法
  
    >
    > **Ps**
    > 先填写好 conf.ini 里的信息DeviceId（设备id）和Key （安全码），然后在运行
    > 建议给可执行文件 777 权限
    >
 
  * [下载](https://github.com/lrqtech/smya-go/releases)可执行文件
    解压&运行./smya
    
  * 自己编译打包
    
    下载本代码
    进入代码目录
  
    >
    > go mod tidy
    > go build
    > 
    
    运行./smya
